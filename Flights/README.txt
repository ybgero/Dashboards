Description:
This dashboard provides an annual overview of flight operations, monitoring key performance indicators related to flight punctuality and cancellations.

Key Metrics:

Total Flights – The number of flights operated throughout the year.
Delayed Flights – The number of flights that experienced delays.
Cancelled Flights – The number of flights canceled.
Percentage Metrics – Proportion of total, delayed, and canceled flights.
Features:

Yearly trend analysis of flight delays and cancellations.
Visual breakdown of flight status for performance monitoring.
Usage:

Useful for airline operations teams, airport management, and analysts tracking flight reliability.
